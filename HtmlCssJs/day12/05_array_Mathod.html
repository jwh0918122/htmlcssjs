배열 관련 메서드 : 유사 배열에서는 사용 제한 되는 경우가 있음

  • join() : 배열 원소 사이에 문자를 삽입하여 새로운 문자열을 생성
  • array.push(value) : 배열의 마지막에 value를 추가 => array[array.length] = value 와 같은 의미 리턴 : 배열의 총 길이를 리턴
  • pop() : 배열의 마지막 요소를 제거 리턴 : 마지막의 요소를 리턴
  • shift() : 배열의 첫 요소를 제거 후 값을 리턴 => pop과 비슷. 위치만 반대
  • unshift(value) : 배열의 가장 앞에 원소를 추가 = 길이를 리턴
